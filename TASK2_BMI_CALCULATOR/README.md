# BMI Calculator (GUI Based)

## Project Description
This project is a graphical Body Mass Index (BMI) Calculator developed using Python and Tkinter. The application allows users to enter their name, weight, and height to calculate their BMI and determine their health category. The project also stores user data and provides history viewing and graphical BMI trend analysis.

## Features
- Graphical User Interface using Tkinter
- BMI calculation using standard formula
- BMI category classification (Underweight, Normal, Overweight, Obese)
- User data storage in a text file
- History viewing option
- BMI trend graph using Matplotlib
- Error handling for invalid inputs
- Clear history option

## Technologies Used
- Python
- Tkinter (GUI)
- Matplotlib (Graph visualization)
- File Handling

## BMI Formula
BMI is calculated using:

BMI = Weight (kg) / Height (m²)
