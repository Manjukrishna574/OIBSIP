#Project Description
This project is a Weather GUI Application developed using Python. The application allows users to enter a city name and get current weather details such as temperature, humidity, wind speed and weather condition.

#Features
 Graphical User Interface using Tkinter
 City based weather search
 Temperature display
 Humidity information
 Wind speed display
 Weather condition display
 Error handling for invalid cities

# Technologies Used
- Python
- Tkinter
- Requests library
- OpenWeatherMap API

# How to Run the Project

Install required library:

pip install requests

Run the program:

python weather.py

# Output
The application displays weather details in a GUI window after entering the city name.
