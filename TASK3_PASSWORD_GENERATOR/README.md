Description
This project is a GUI-based password generator developed using Python and Tkinter. It allows users to create secure and customizable passwords by selecting different character types and specifying the desired length. The application also evaluates the strength of the generated password.
Features
Generate secure passwords using the secrets module
Customize password length (6 to 40 characters)
Select character types:
Lowercase letters
Uppercase letters
Numbers
Symbols
Real-time password strength indicator (Weak, Medium, Strong)
Copy password to clipboard
Simple and user-friendly interface
